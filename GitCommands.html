Git Commands:

  • git status

    • To check the status of which files that we done changes.

  • git add .

    • To add all the files that we done changes.

  • git add (file_name.exe)

    • To add only specify file out of all changed files.

  • git checkout -b (branch_name)

    • To create new branch and focus is into that created branch.

  • git checkout (branch_name)

    • To swift into that specific branch.

  • git log

    • To see the timestamps/logs of a file/folder that we created/modefied

    Note: To exit from this logs, press "Q"

  • git diff

    • To see what exact changes in a file that we added

  • git commit -m (commit_message)

    • To commit the added file to GitHub server with commit message/saving a version of changes.

  • git pull

    • To take latest pull from the GitHub.

  • git push

    • To push the committed changes/files into the GitHub.

  • git merge

    • To merge 2 brances (doing this will helps in overcome if any commit markers).

    Note: To exit from this merge details, press ":Q"

  • git reset --hard (commit_ID)

    • To revert/reset the currect changes to purticular past commit.
